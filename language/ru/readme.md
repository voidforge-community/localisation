# Переменные которые применяются независимости от полей

| Тип данных | Имя переменной             | Описание                                                 | Пример вывода |
|------------|----------------------------|----------------------------------------------------------|---------------|
| Текстовое  | %current-time%             | Выводит время в формате ЧЧ:ММ:СС (Пример 12:55:39)       | 12:55:39      |
| Текстовое  | %version:BOT%              | Выводит версию бота                                      | 2.0.0-beta.02 |
| Текстовое  | %version:FUN_CALC_ENGINE%  | Выводит версию движка калькулятора                       | 0.2           |
| Текстовое  | %version:LIB_JDA%          | Библиотека для упрощенного доступа к API discord         | 6.1.0         |
| Текстовое  | %version:JAVA%             | Язык программирования                                    | 17            |
| Текстовое  | %version:GITHUB_API%       | Библиотека для упрощенного доступа к API github          | 1.101         |
| Текстовое  | %version:LAVALINK_CLIENT%  | Для подключению и работы с LavaLink Play                 | 3.2.0         |
| Текстовое  | %version:SPARK_CORE%       | RestAPI                                                  | 2.9.3         |
| Текстовое  | %version:JAVA_WEBSOCKET%   | Подключение к внешним приложениям через WebSocket        | 1.5.3         |
| Текстовое  | %version:SQLITE_JDBC%      | Подключение к БД Sqlite через JDBC                       | 3.44.1.0      |
| Текстовое  | %version:MYSQL_CONNECTOR%  | Подключения к БД MySQL через JDBC                        | 8.0.33        |
| Текстовое  | %version:GSON%             | Для парсинга JSON (В ближайшее время будет удалена)      | 2.8.9         |
| Текстовое  | %version:SLF4J_API%        | Библиотека                                               | 1.7.32        |
| Текстовое  | %version:SNAKEYAML%        | Обработка YAML файлов                                    | 2.0           |
| Текстовое  | %version:JLINE%            | Библиотека                                               | 3.21.0        |
| Текстовое  | %version:LOGBACK_CLASSSIC% | Библиотека                                               | 1.2.11        |
| Текстовое  | %version:XXX%              | Вернет переменную текстом где XXX неизвестная переменная | %version:XXX% |
| Текстовое  | %version%                  | -                                                        | unknown       |

## Формат записи
```yml
system:
  # Секция системных сообщений
  no-permission: # Сообщение, показываемое при отсутствии прав у пользователя
    embed: # Структура в формате Embed (встраиваемое сообщение Discord)
      title: "403: Отказано в доступе" # Заголовок эмбеда
      description: | # Основной текст эмбеда (многострочный)
        При использование данной команды произошла ошибка — у пользователя нехватает прав!
        Требуемые права: `%hasPermission%` # Плейсхолдер для вывода нужных прав
      footer: "Время вызова: %current-time% | VoidBot %version% :3" # Нижняя подпись, где %current-time% — текущее время, %version% — версия бота
```
## В зависимости от строк есть разные переменные
Указывать их все смысла нет что они выдают можно понять из названия :3

## Нужны еще ключи? или ключ не работает
Рядом с ключом с текстом просто напиши комментарий пример
```yml
system:
  # Ошибка если нет прав
  no-permission: 
    embed:
      title: "403: Отказано в доступе"
      # Переменная %hasPermission% не работает
      description: |
        При использование даннной команды произошла ошибка у пользователя нехватает прав!
        Требуемые права: `%hasPermission%`
      # Тут хотелось бы еще добавить в универсальные переменные вывод даты
      footer: "Время вызова: %current-time% | VoidBot %version% :3"
```
